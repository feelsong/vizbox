# vizbox
![Screen Shot 2021-08-14 at 9 28 57 PM](https://user-images.githubusercontent.com/5912986/129467204-4f863f10-9ea9-429e-a485-5e673d90f30b.png)
 A collection of data visualizations with `d3.js`, `c3.js` and `angular.js`.

#How To Develop
1. Start with `generator-gulp-angular`
2. Add `d3` & `c3`
3. Add `topojson`
4. Create directives

#How To Run
1. `gulp serve` to run.
